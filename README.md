# VRCSupport
VRChat Unity Support Plugin

## Feature
 - Emote On/Off 구현 자동화
 - AnimationClip 합치기
 - Missing Animation 원클릭 제거
 
## Future
 - AudioClip FFT Visualizing Generator

## How to

### Install
1. Import [VRCSupport.unitypackage](https://github.com/SteaI/VRCSupport/releases)
2. Unity > 5ちゃ★ > VRCSupport
3. Enjoy!
